# Cancer-pred

Brain tumor detection using deep learning and TensorFlow. This project includes data preprocessing, model building, training, and medical image prediction in Jupyter Notebooks. Useful for students and researchers exploring medical AI.

## Features
- Data loading and preprocessing
- Neural network model training
- Tumor prediction from images
- Well-commented Jupyter notebooks for easy study

## Getting Started
1. Clone the repository
2. Install dependencies (TensorFlow, Jupyter, etc.)
3. Open and run the notebooks

## Project Files
- `model.png` - Model architecture visualization
- `piko.ipynb` - Main notebook for data exploration
- `Prototype.ipynb` - Prototype experiments
- `TensorflowTensors.ipynb` - TensorFlow tensor operations
- `todo.txt` - Project tasks
